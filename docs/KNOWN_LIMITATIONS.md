# Known Limitations

DesktopVisual 1.0.0 is scoped to authorized visible-first Windows desktop automation. It is not a universal autonomous IDE developer.

Limitations:

- Visual Studio C++ multi-file workflow is not included in 1.0.0.
- Complex arbitrary app and web automation remains experimental.
- Complex IDE workflows need explicit target windows, goals, constraints, and user review.
- The binary is unsigned.
- Protected desktops, elevated windows, credential prompts, CAPTCHA, proctoring, anti-cheat, and anti-automation systems are stop conditions.
- The public distribution does not include source code or build project files.
