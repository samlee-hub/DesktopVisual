# Known Limitations

DesktopVisual is a local Windows desktop automation runtime. It does not guarantee compatibility with every app, browser state, website, accessibility tree, or security environment.

This package does not bypass CAPTCHA, human verification, credential or account verification, proctoring, lockdown browser, anti-cheat, anti-automation, or third-party security controls.

The default/full_access user mode selector is deferred. Limited access mode and per-task permission mode are also deferred.

This package is a release candidate, not a promise of broad compatibility with protected or restricted environments.
