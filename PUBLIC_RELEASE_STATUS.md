Release strategy:

This public release is a developer-mirror public release.

The previous permission-restricted public release strategy is archived for now.

This release keeps the same functionality, runtime capability, permission mode, developer profile behavior, source code, tests, adapters, Skill templates, documentation, and verification files as the developer version.

Only the release directory structure, public entrypoint, local installed Skill target, and public documentation are organized for normal use and GitHub publication.

No permission reduction is applied in this release.

No open-source / closed-source split is maintained in the current line.

The current priority is stable local use, repeatable visible desktop execution, recording readiness, and full functional support.

Permission reduction may be reconsidered only after the public release is stable enough for normal use and recording.
