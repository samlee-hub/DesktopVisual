# DesktopVisual 1.0.0 Release Notes

DesktopVisual 1.0.0 is the first closed-source public binary distribution derived from the frozen developer baseline.

## Included capabilities

- Visible-first Windows desktop automation runtime.
- Real mouse and keyboard actions through user-level Windows input.
- Global DPI-aware screenshots.
- Target window lock and coordinate mapping.
- Foreground preempt/cache support.
- Operation timeline profiling.
- Structured visible text input.
- Public-safe command protocol, safety documentation, and Codex skill.

## Not included

- Source code.
- Visual Studio project files or C++ source files.
- Developer logs, raw artifacts, runtime sessions, screenshots, installers, or intermediate build outputs.
- Visual Studio C++ multi-file workflow acceptance.

## Binary

The Windows binary is unsigned. Verify `checksums/SHA256SUMS.txt` before use.
