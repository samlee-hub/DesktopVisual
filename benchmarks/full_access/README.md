# Full Access Benchmark Harness

This directory contains deterministic local scenarios for v3.3.10 FULL_ACCESS evidence.

The harness uses local fixtures, TestWindow, policy dry-runs, and simulated FULL_ACCESS session files. It does not use real accounts, real communication services, browser profiles, raw motion data, or release artifacts.

Run:

```powershell
D:\desktopvisual\full_access_benchmark_matrix.ps1
D:\desktopvisual\full_access_benchmark_selftest.ps1
D:\desktopvisual\export_full_access_evidence_pack.ps1
```
